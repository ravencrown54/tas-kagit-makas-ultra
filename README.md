# ✊ Taş • Kağıt • Macera — Ultra Edition ✋✌️

C# Windows Forms teknolojisi kullanılarak geliştirilmiş, modern tasarımına ve görsel geri bildirimlere sahip arcade tarzı bir **Taş-Kağıt-Makas** oyunu.

---

## 🚀 Özellikler

* **Modern Arayüz Tasarımı:** Gözü yormayan şık, karanlık renk paleti.
* **Dinamik Arka Planlar:** Oyunun gidişatına göre (Kazanma durumunda yeşil, kaybetmede kırmızı, beraberlikte altın sarısı tonlarında) renk değiştiren dinamik form tasarımı.
* **Görsel Geri Bildirimler:** Sen ve bilgisayar hamle yaptığınızda ekranda beliren dev emojiler sayesinde anlık durum takibi.
* **Skor Tablosu & Sıfırlama:** Sürekli güncellenen skor takibi ve skorları tek tıkla sıfırlayabileceğiniz bir sıfırlama butonu.
* **Sıfır Hata & Güvenli Kod:** .NET'in yeni nullable (boş değer uyarısı) standartlarına tam uyumlu, optimize C# altyapısı.

---

## 🛠️ Kullanılan Teknolojiler

* **Programlama Dili:** C# 
* **Arayüz Altyapısı:** .NET 10.0 Windows Forms
* **Geliştirme Ortamı:** Visual Studio Code

---

## 📦 Kurulum ve Çalıştırma

Oyunu bilgisayarınızda yerel olarak çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

### 1. Gereksinimler
Sisteminizde [.NET SDK](https://dotnet.microsoft.com/download) yüklü olmalıdır.

### 2. Projeyi Klonlayın veya İndirin
```bash
git clone [https://github.com/ravencrown54/tas-kagit-makas-ultra.git](https://github.com/ravencrown54/tas-kagit-makas-ultra.git)
cd tas-kagit-makas-ultra

```

### 3. Oyunu Başlatın

Proje klasörünün içerisindeyken terminale şu komutu yazarak oyunu doğrudan başlatabilirsiniz:

```bash
dotnet run

```

---

## 🎮 Nasıl Oynanır?

1. Oyunu başlattığınızda karşınıza modern siyah bir oyun paneli gelecektir.
2. Alt kısımda yer alan **✊ TAŞ**, **✋ KAĞIT** veya **✌️ MAKAS** butonlarından birine tıklayarak hamlenizi yapın.
3. Yapay zeka anında rastgele bir karşı hamle yapacak ve sonuç ekranınızda belirecektir.
4. Skorları sıfırlamak ve en baştan başlamak istediğinizde alttaki **🔄 Skorları Sıfırla** butonunu kullanabilirsiniz.
